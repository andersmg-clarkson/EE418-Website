Reference Course

EE418 Software Engineering Senior Design

Study of the principles and practices of software engineering. Topics include software engineering tools, process models, software requirements analysis, design methodologies, software testing, and software maintenance. Hands-on experience building a software system using the waterfall life cycle model and multiple software engineering tools. Working in a team environment, students will develop all life cycle deliverables: requirements document, design documents, test plan and test suites, source codes, and user manuals
